Planche-Contact — Manuel complet

1. Présentation

Planche-Contact est un outil léger et rapide permettant de générer des planches contact photographiques de haute qualité (300 dpi) sans utiliser de base de données ni de logiciel lourd.

Il existe sous deux formes :

  • Interface graphique GTK4 (planche-contact-gtk.py)
  • Ligne de commande (portfolio.py)

2. Interface Graphique (GTK)

2.1 Onglet « Génération »

Champ / OptionDescription
Titre du projetTexte affiché en grand en haut de chaque planche.
Nom de l’auteurNom affiché en dessous du titre (en gras et plus gros).
Dossier d’entréeDossier contenant les photographies à traiter (obligatoire).
Dossier de sortieDossier où seront créés les résultats. Par défaut : Portfolio/ dans le dossier source.
Recherche récursiveSi coché, analyse aussi tous les sous-dossiers.
Images par plancheNombre de vignettes par page (9, 12, 16, 20 ou 25).
Format de la plancheFormat papier : A5, A4, A3, A2 ou Letter.
Filigrane (texte)Texte qui sera répété sur toutes les images (ex: © Ton Nom).
Orientation du filigrane
  • Horizontal : texte horizontal
  • Diagonale horaire : texte incliné à ~32°
  • Diagonale anti-horaire : texte incliné à ~-32°
Options d’exportPDF, Galerie HTML paginée, Index CSV.

2.2 Résultats générés

  • planches/ → Images JPEG individuelles de chaque planche
  • portfolio.pdf → PDF complet de toutes les planches
  • gallery/ → Galerie HTML auto-contenue (avec miniatures)
  • index.csv → Liste des fichiers avec leur numéro

3. Interface en Ligne de Commande (CLI)

3.1 Options principales

OptionDescriptionExemple
-i, --inputDossier source (obligatoire)-i /Photos/Voyage
-o, --outputDossier de sortie-o ./Resultats
-n, --numNombre d’images par planche-n 16
--formatFormat de page--format A3
--titleTitre du projet--title "Voyage 2025"
--authorNom de l’auteur--author "Abel GEZ"
--watermarkTexte du filigrane--watermark "© Abel GEZ"
--watermark-orientationOrientation du filigrane--watermark-orientation "Diagonale horaire"
--pdfGénérer un PDF
--htmlGénérer une galerie HTML
--csvGénérer un index CSV
-r, --recursiveAnalyse récursive

3.2 Exemples complets

# Génération simple
python3 portfolio.py -i /Photos/2025 -o ./Sortie --pdf --html

# Version professionnelle avec filigrane diagonal
python3 portfolio.py \
  -i /Photos/Islande \
  -n 16 \
  --format A3 \
  --title "Islande 2025" \
  --author "Abel GEZ" \
  --watermark "© Abel GEZ" \
  --watermark-orientation "Diagonale horaire" \
  --pdf --html --csv

4. Recommandations

  • Pour un rendu homogène, conservez le même nombre d’images par planche sur tout le projet.
  • Le filigrane en diagonale est généralement plus élégant et moins gênant visuellement.
  • La galerie HTML est autonome (vous pouvez la copier ailleurs sans rien casser).
